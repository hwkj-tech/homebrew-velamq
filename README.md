# Homebrew Tap for VelaMQ

This is the Homebrew tap for VelaMQ.

This tap is installed from GitHub. It does not own a GitHub Pages custom
domain; release artifacts are downloaded from `https://repo.velamq.com/velamq`.

## Install

```bash
brew tap hwkj-tech/velamq
brew install velamq
```

## Service

```bash
brew services start velamq
brew services restart velamq
brew services stop velamq
```

## Release Sync

The `Formula/velamq.rb` file is generated by the release scripts in
`velamq-rs` after macOS artifacts are built:

```bash
cd /path/to/velamq-rs

DOWNLOAD_BASE_URL=https://repo.velamq.com/velamq/downloads \
  bash deploy/release/update-homebrew-formula.sh
```

Set `SYNC_PUSH=1` when the generated commit should be pushed.

Homebrew maps:

```text
brew tap hwkj-tech/velamq
```

to:

```text
github.com/hwkj-tech/homebrew-velamq
```
